# convertPropertyNameFromLowerCase( Common\string $name )

Преобразует имя свойства из lowerCamelCase в нотацию, принятую в целевом классе. Может быть переопределен в целевом классе. По умолчанию предполагается, что нотация, принятая в целевом классе - это lowerCamelCase, так что метод возвращает имя свойства без изменений.
